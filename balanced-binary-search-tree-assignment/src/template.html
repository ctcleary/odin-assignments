The Odin Project - Balanced Binary Search Tree Assignment

All the magic is in the console.

Create a new balanced BST with:

new Tree([number, number, .... ])

And see the result printed in the console. Which will also show the "pretty printed" results of:

new Tree([1, 7, 4, 23, 8, 9, 4, 3, 5, 7, 9, 67, 6345, 324])

The numbers will be sorted, and duplicates removed (as per the assignment spec).
